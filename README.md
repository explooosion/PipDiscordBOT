## Installation

You can create bot to your server.

```bash
git clone https://github.com/explooosion/PipDiscordBOT.git
```

```bash
cd PipDiscordBOT
```

```bash
npm install
```

## Usage

1. Copy file `.env.example` and rename to `.env`,  
2. Set the `DISCORD_TOKEN` 

```bash
npm start
```
